Project Description:

Employee Productivity Prediction is a machine learning-based web application that predicts the actual productivity of garment workers based on various input features like team, day, SMV, incentives, and more. This platform uses advanced models like XGBoost, Random Forest, and Linear Regression for accurate predictions and is integrated with a Flask web app for seamless interaction.
